# Smart Home Dashboard

A Flutter app to manage and monitor smart devices like lights, fans, ACs, and cameras in different rooms. Features a modern, interactive UI with toggle switches and device cards.

## Features

- Dashboard view of all devices
- Toggle devices ON/OFF
- Add new devices with type and room
- Modern Material 3 UI with shadows and gradients
- Responsive and interactive design
